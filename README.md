# 🎤 Interview Assistant

An **AI-powered Interview Assistant** that simulates real interviews and provides detailed feedback.

## ✨ Features
- 🎯 **Domain-specific questions** (50+ roles)
- 🤖 **AI scoring** with rubric evaluation
- 📋 **STAR method checker** for behavioral questions
- 🔊 **Voice-to-text** support
- 📊 **Performance dashboard**

## 🛠️ Tech Stack
- **OpenAI GPT-4** – question generation & evaluation
- **Whisper API** – speech recognition
- **LangChain** – conversation management
- **FastAPI** – backend server
- **React / Streamlit** – frontend

## 🚀 Getting Started
```bash
git clone https://github.com/Varshini487/interview-assistant
cd interview-assistant
pip install -r requirements.txt
streamlit run app.py
```

## 💡 Use Cases
- Job interview preparation
- Campus placement training
- Mock interview platforms
- Corporate training programs
